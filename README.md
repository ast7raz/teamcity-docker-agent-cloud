# Tomych Teamcity agent for cloud builds

Teamcity agent image based on https://hub.docker.com/r/jetbrains/teamcity-agent/ with following additions:

1. Python 3.6, pip3
2. tzdata, curl, unzip
3. aws-cli and ecs-cli
